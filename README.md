## Featured work

### DesignRail
A design-system handoff control plane for AI-assisted implementation. It maps Figma-style component intent to Shoelace Web Components, runs compliance review, records human decisions, and exports implementation-ready examples.

### zod-ai-tool
A small TypeScript package for defining AI tools once with Zod, deriving Anthropic/OpenAI/Gemini tool definitions, and validating model tool input with the same schema.
